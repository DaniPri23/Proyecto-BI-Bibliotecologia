# Proyecto BI - Transición de Planes de Estudio

## Organización
Escuela de Bibliotecología y Ciencias de la Información.

## Problema
Analizar la transición de estudiantes de los planes actuales hacia los nuevos planes de estudio 2027.

## Herramientas
- draw.io
- Python
- Power BI
- Excel
- Word
- Canva
- SQLlite

## Estructura del repositorio
- docs: documentación del proyecto
- data: fuentes de datos y datos procesados
- etl: scripts de transformación
- database: scripts SQL
- dashboard: solución analítica
- diagrams: modelos de datos
- evidence: evidencias de trabajo
